Lesson 2: Creating a Structured Document with HTML

HTML and CSS are both "languages".HTML controls the "structure" of a web page, CSS controls the "style" of a page (how it looks). When programmers talk about the "DOM" they are talking about the tree-like structure of a page.Everything on a web page is actually a box. In order to put boxes on a web page, "div" tag is used. When exparts consider the layout of a webpage, they start from allocating big boxes like title, body, footer, then put small boxes inside of them. Utilizing "class" is useful when define the style of div tags. In order to identify divs you make, give appropriate name for each divs as class name is recommended.
